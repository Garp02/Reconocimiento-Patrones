\subsection{Implementación de LBM}

Para la implementación de este método se optó por hacer métodos auxiliares para 
tener mayor control ante errores; además, gracias a estos métodos auxilires se 
puede modificar la implementación según las necesidades de quien lo use.

También se implementó un método que crea histogramas normalizados con 256 bins 
de una imagen a la que ya se le aplicó LBP. Esto para poder imprimir la imágenes 
procesadas con su histograma de frecuencias y su patrón LBP de 0 a 255.

Se mostrarán imágenes con sus respectivos histogramas mediante iteraciones sobre la 
lista de imágenes; para esta lista tomamos las etiquetas únicas (etiquetas por clase) 
y con ello podemos traer todas las clases del dataset de imágenes.

\subsection{Extracción de características}

Para extraer las características del LBP usamos una lista e iteramos las imágenes 
en la lista de las imágenes del dataset. Invocamos al método que implementa el lbp 
y el histrogramanormalizado, después, agrgamos el patrón del LBP a la lista de 
características. Este proceso tomó una media de tres minutos y medio. 

Después de extraer las características, las pasamos a un arreglo \texttt{X} para 
poder hacer el entrenamiento del modelo supervidado \texttt{K-NN}, para ello 
también necesitamos tomar las etiquetas de en otro arreglo \texttt{y}.

Ahora dividiremos los conjuntos de prueba y entrenamiento con \texttt{train\_split\_test}, 
para ello asignaremos un 30\% de los datos a prueba y el otro 70\% a entrenamiento. 
Para probar nuestro modelo supervisaro iniciaremos el entrenamiento con 3 vecinos y con la 
métrica euclidiana. Así obtenemos la siguiente tabla:
