Para la carga de datos iteramos sobre la carpeta \texttt{KTH\_TIPS} buscando
las carpetas y leemos solo los archivos que sean imágenes; después se  convierte
la imagen a otra paleta de colores y se redimencionan a 200 px $\times$ 200 px.
Después las guardamos en un array de numpy y calculamos los archivos encontrados
(con la longitud del array).

Para comprobar que los datos hayan sido cargados correctamente, tomamos un
elemento \texttt{im} del array para visualizar la imagen en escala de grises
