Implementar el descriptor \textbf{Patrones binarios locales} en el
código para extraer características de texturas y clasificarlas
utilizando \texttt{K-NN}. El proceso incluye:

\begin{itemize}
    \item[$\alpha_1.$] Cálculo de mapa de patrones binarios locales.
    \item[$\alpha_2.$] Extracción de histogramas de características.
    \item[$\alpha_3.$] Clasificación y evaluación de resultados.
\end{itemize}
